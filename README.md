# Tic Tac Toe

Un semplice gioco Tic Tac Toe sviluppato per esercitarsi con la logica di programmazione e la gestione degli eventi.

## Funzionalità

- Modalità 2 giocatori
- Controllo automatico del vincitore
- Pareggio automatico
- Pulsante di reset
- Interfaccia semplice e responsive

## Tecnologie utilizzate

- HTML
- CSS
- JavaScript

## Installazione

1. Clona il repository:

```bash
git clone https://github.com/tuo-username/tic-tac-toe.git

```
2. Apri la cartella del progetto:

```bash
cd tic-tac-toe
```

3.Apri index.html nel browser.

Come giocare

Il giocatore X inizia per primo.
I giocatori si alternano cliccando sulle caselle.
Vince chi completa una riga, colonna o diagonale.
Se tutte le caselle sono occupate senza vincitore, la partita termina in pareggio.
